# ✝️ Church Website

A modern and responsive church website built to provide information about services, events, and community engagement.

## 🌐 Live Demo
🔗 https://church-website-two-ecru.vercel.app/

---

## 📌 Overview

This project is designed to create a clean, user-friendly experience for visitors looking to explore church activities, sermons, and announcements.

It focuses on:
- Simple navigation
- Mobile responsiveness
- Clean UI/UX
- Fast performance

---

## 🚀 Features

- 📱 Fully responsive design
- 🗓️ Events & announcements section
- 📖 Sermons / content sections
- ⛪ Church information pages
- ⚡ Fast loading and optimized performance
